# I Built Interview Memory With Hindsight #

## 🚀 Project Description

** I Built Interview Memory With Hindsight** is an intelligent interview preparation platform that helps students and job seekers prepare for technical and HR interviews through personalized AI-powered coaching.

Unlike traditional interview platforms, our system remembers previous interview sessions, tracks weak topics, and continuously adapts to each user's learning journey. By leveraging **Hindsight** for persistent memory and **cascadeflow** for intelligent runtime routing, the platform delivers smarter, faster, and more personalized interview preparation.

Whether you're preparing for your first internship or your dream software engineering role, the platform provides realistic mock interviews, coding assessments, resume-based questions, and actionable AI feedback.

# ⚙️ Installation Steps

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/ai-interview-career-coach.git
```

## 2. Navigate into the Project

```bash
cd ai-interview-career-coach
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Configure Environment Variables

Create a `.env` file and add:

```env
OPENAI_API_KEY=your_api_key
HINDSIGHT_API_KEY=your_hindsight_key
DATABASE_URL=your_database_url
```

## 5. Start the Development Server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
